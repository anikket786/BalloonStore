# Instructions to run the app

1. Open the BalloonStore Directory inside Android Studio and let the gradle sync finish running.
2. Once gradle sync is completed, just connect the android emulator or physical device to your computer.
3. Run the app by clicking on Run button at top menu bar within Android Studio.
4. Let the build and installation process to complete once completed it can be tested.

# App architecture & libraries used

1. Follows MVVM architecture
2. Uses ViewBinding
3. Apollo for communicating with graphql server
4. Room library for Local database
5. Paging 3 library Pagination
6. Daqger Hilt for Dependency Injection
7. Jetpack Navigation component
